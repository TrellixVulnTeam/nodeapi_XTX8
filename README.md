<h1>Created USER CRUD rest API with typescript + Express.js + Sequelize + Sqlite3</h1>

## PORT

```
9000
```

## Installation

```
yarn install
```

## Run locally

```
yarn run dev
```

## Build

```
yarn run build
```

## Core functionality

- [] set up typescript for dev
- [] basic express server with typescript
- [] setup database
- [] create model
- [] create user crud api
- [] read user with pagination
- [] read user with id
- [] update user
- [] delete user
